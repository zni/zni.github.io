Some Things I've Worked On

(In recent hobby seasons)

xmachine

The idea was to write an emulator of some kind, but I eventually wanted to turn it into a PDP-11 for some reason.

So it's basically PDP-11 assembly, and some PDP-11-like components. It has a disk controller of sorts to read and write to disk images.

  • https://github.com/zni/xmachine


fslib

So this was a continuation, or final completion, of the goal of getfat. That goal was to write a program to read and write to FAT32 images.

The extended goal, which I haven't finished and will probably come back to, is to support multiple file system formats. I haven't started on that part, but this seems like a good hobby season for that.

  • https://github.com/zni/fslib
